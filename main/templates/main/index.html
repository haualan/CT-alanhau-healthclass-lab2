Selected Projects

Keyword Extractor from Unstructured Text



A Fortune 100 bank approached our student team to solve a particular problem about their internal search engine. The institutional sales team was having trouble finding relevant research reports for their clients because research analysts had difficulty adding keywords to reports when they were written. Due to the sheer volume and length of each report (>500,000 new reports per year from two to hundreds of pages) it's manually infeasible to tag each of them by hand.

Using Named Entity Recognition (NER) and Term Frequency/ Inverse Document Frequency (TF-IDF) models wrapped with Python's Flask library, we have developed a web accessible API hosted with Apache on a simple Linux cloud server that allows keywords to be extracted from any unstructured text.

Not only can new reports be tagged automatically, the API can be scaled to run retroactively on millions of old reports in the bank's database.

Quantitative Scoring using NLP on Patient Centered Documents

Content creators (of websites / articles / pamphlets / handouts) face challenges of writing engaging materials for their readers, especially in a healthcare context. Using a "bag of words" and n-grams approach I have trained a model against the helpfulness vote on a medical forum targeted on patients with Rheumatoid Arthritis (RA).

The result is a model that accepts unstructured text as an input and outputs a score from 0 to 100 to gauge whether the writing is engaging for readers looking for information on RA.

Training on different contexts can potentially yield sentiment and other classification models for a wide variety of uses in many industries.


CarePigeon: Platform for Physicians to Reduce Surgical Risk and Liability

Previous Next

Over 3 Billion dollars is paid by insurance companies every year to settle lawsuits arising from medical malpractice on elective surgeries. Most of them are due to miscommunication arising from gaps in expectation and a lack of effort spent on patient education.

CarePigeon uses a MOOC-like approach (think Coursera, Khan Acadmemy...) to track the learning progress and compliance of a patient and streamlines the informed consent signing process. Surgeons can opt-out patients until they truly understand the risks, benefits and alternatives to their operation. This protects patients from having operations before they are emotionally ready and surgeons from unwanted liability.

Etrade Python API

Applying machine learning methods to investing could be a fun thing to do, but there are barriers for investors even if they are well versed in Python. It is quite difficult to implement SciPy or NumPy models when most of the brokerage houses only provide APIs with Java or C++ examples. This is an open source library I've written that allows easy integration with Etrade's API for those investors who are looking to write programs in Python.
